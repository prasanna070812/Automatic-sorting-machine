# 🔧 Automatic Sorting Machine

## 📌 Overview
This project is an automatic machine designed to separate metals and non-metals efficiently.

## 🎯 Features
- Portable and lightweight
- Budget-friendly (~₹3000)
- Automatic sorting
- Display for count/weight
