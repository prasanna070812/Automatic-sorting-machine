# Need statement:
Design an Automatic sorting machine, which would help to easily separate Metals and Non-metals. The machine should be easily portable and user friendly. The machine must not exceed the size of 2ft*2ft*2ft.The budget to build machine should be around 3000 and the weight of machine should be around 2kg.It can be multicolored with manual display system. It must have an on-off indications and also it has to keep an count of weight of products sorted .Ultimately it must be capable of working in small junkyards and scrapyards.
